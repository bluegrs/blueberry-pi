# Blueberry-Pi
Mobile gesture recognition glove to send hand orientation data over Wi-Fi network.

## Hardware
Raspberry Pi 3 B+

LSM303DLHC: 3-axis accelerometer and 3-axis magnetometer

L3GD20: 3-axis gyroscope
